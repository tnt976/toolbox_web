# 龙华滢水山庄1区房屋编码管理

本项目提供了龙华滢水山庄1区房屋编码的查询工具和表格模板。

## 📁 文件说明

- `house_code_query.py` - 房屋编码管理工具（Python脚本）
- `滢水山庄一区房屋编码表.csv` - 完整的房屋编码表格模板（CSV格式，1344条记录）
- `滢水山庄一区房屋编码表.json` - 完整的房屋编码表格模板（JSON格式）
- `房屋编码表示例.csv` - 简化的示例表格
- `房屋编码表.html` - 美观的HTML表格页面

## 🔍 查询房屋编码的方法

### 方法一：官方网站查询
1. 访问 [深圳统一地址查询系统](https://spatydz.sz.gov.cn/web/index.html#/)
2. 在搜索栏输入 "滢水山庄一区"
3. 选择最准确的地址结果
4. 进入基础信息后，点击【楼栋结构】
5. 查找您需要的楼层和房间号，即可看到25位数的房屋编码

### 方法二：其他方式
- **房门二维码**：查看房门上的二维码贴纸，直接扫码获取
- **网格员咨询**：在楼栋公告栏找到网格员联系方式咨询
- **社区工作站**：直接到房屋所在社区工作站咨询
- **证件查询**：查看房屋租赁凭证、房产证等证件

## 📊 使用表格模板

1. 打开 `滢水山庄一区房屋编码表.csv`（可用Excel或WPS打开）
2. 在官网查询到房屋编码后，填入对应的单元格
3. 保存文件即可

## 🏘️ 小区信息

- **小区名称**：滢水山庄一区
- **位置**：龙华区梅林关附近，梅坂大道91号
- **楼栋数**：约28栋
- **总户数**：约1246户
- **建成年份**：1998年左右
- **建筑类型**：多层楼梯房

## 💡 提示

- 房屋编码为25位数字
- 楼栋编码为19位数字
- 请确保填写的编码准确无误
